Atmosphereic & Weather Measurements by Geo-Latitude

For this project, weather changes were analyzed based on distance from equator (latitude). 500 cities' weather data was pulled via the OpenWeatherMap API. Matplotlib visulizations were created to represent changes in temperature, cloudiness, wind speed, and humidity. These visualizations, comparisons, and the raw data set are included herein (see links above).
